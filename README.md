[![Python Version](https://img.shields.io/badge/python-3.11%20%7C%203.12-blue.svg)](https://www.python.org/)

**Astro-Guard-Module** (Star-Force Sentinel) is a high-performance 2D arcade shooter developed as part of my **2025 Legacy Project Series**. It demonstrates optimized sprite management and real-time collision physics using Python.

## 🛠️ Key Features
- **Optimized Rendering:** Uses Arcade's `SpriteList` for high FPS.
- **Physics Engine:** Custom logic for projectile interception and bounce mechanics.
- **Clean Architecture:** Fully Object-Oriented (OOP) structure.

## 🎮 How to Run
1. Ensure you have **Python 3.11 or 3.12** installed.
2. Clone the repo: `git clone https://github.com/Nosrat-Jahan/Astro-Guard-Module.git`
3. Install dependencies: `pip install -r requirements.txt`
4. Run the module: `python AstroGuard.py`
